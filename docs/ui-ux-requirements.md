## Chart Display

### 宮格佈局
- [x] 4×3 九宮格標準排法
- [x] 中間 2×2 為中心資訊區

### 每個宮格需顯示
- [x] 宮名（命宮、兄弟…）
- [x] 天干地支（如戊寅）
- [x] 主星（major）
- [x] 輔星（minor/soft）
- [x] 雜曜／煞星（adjective/tough）
- [x] 亮度標記（廟、旺、得、利、平、不、陷）
- [x] 四化標記（祿、權、科、忌，顏色區分）
- [ ] 大限數字（在中心區或宮格內顯示）（已在運限面板以彈性方式呈現）

### 中心資訊區（4格）
- [x] 五行局（如水二局）
- [x] 命主（廉貞、破軍等）
- [x] 身主（如火星、天同）
- [x] 身宮位置（如身(申)）
- [x] 四柱顯示（B格式）：`庚午年 壬午月 辛亥日 庚寅時`
- [x] 農曆生日（如一九九〇年五月廿三）
- [x] 生肖（如馬）
- [x] 星座（如雙子座）

### 星曜呈現方式
- [x] 文字粗細區分主輔（主星 bold，輔星 normal，雜曜 lighter）
- [x] 亮度標記緊跟在星名後
- [x] 四化以顏色標示（祿=綠色、權=藍色、科=紫色、忌=紅色）

## Fortune Limits Display

### 需要顯示的運限
- [x] 大限（10年）
- [x] 小限（1年）
- [x] 流年
- [x] 流月
- [x] 流日
- [x] 流時

### 呈現方式
- [x] Checkbox 切換式（參考 myfate 行為）
- [x] 勾選後在對應宮位內顯示該層運限資訊
- [x] 未勾選時隱藏，保持版面乾淨

### 運限顯示內容
- [x] 大限：全部12宮位均顯示對應大限範圍，當限宮粗體標示天干地支（如癸未 34-43歲）
- [x] 小限：全部12宮位均顯示該宮小限年齡清單（如「1, 13, 25, 37, 49, 61, 73, 85, 97, 109歲」），以逗號分隔，12年一循環，上限至110歲。當前小限所在宮位粗體標示。
- [x] 流年：全部12宮順時針循環顯示，起始宮顯示完整「流年 丙午」，其餘11宮僅顯示角色名稱（命宮→兄弟→夫妻…），顏色紫
- [x] 流月：全部12宮順時針循環顯示，起始宮顯示「流月 壬辰」，其餘11宮僅顯示角色名稱，顏色青
- [x] 流日：全部12宮順時針循環顯示，起始宮顯示「流日 丙子」，其餘11宮僅顯示角色名稱，顏色橙
- [x] 流時：全部12宮順時針循環顯示，起始宮顯示「流時 戊子」，其餘11宮僅顯示角色名稱，顏色粉紅
- [x] 各層無分隔線，行距緊湊（margin-top:2px, padding-top:1px）
- [x] 運限方向：流年／月／日／時均為順時針（不分性別）

## Prediction Target Time（預測目標時間）

- [x] 日期輸入（YYYY-MM-DD）
- [x] 12時辰下拉選單（與出生時間一致）
- [x] 時辰顯示（如 11:00 → 午時）
- [x] 變更時自動重新計算所有運限

### 換日規則
- 採用 **子正換日**（午夜 00:00 為界）：夜子時屬當天，早子時屬隔天
- 用戶選的日期即為正確的陽曆日期，iztro 內部自動處理夜子時的日柱校正（dayFix）
- 不會對用戶選的日期做額外調整

### Checkbox 清單
- [x] 大限
- [x] 小限
- [x] 流年
- [x] 流月
- [x] 流日
- [x] 流時
- [x] 四化（切換顯示/隱藏所有星曜上的四化標記）
- [x] 自化（切換顯示/隱藏各宮天干對應的自化標記，如自祿、自權）
