# zi-wei-pai-pan

紫微斗數排盤工具 — 純前端 HTML，零後端依賴。

## 功能

### 十二宮星盤
輸入生日（陽曆/農曆）+ 出生時間 + 性別，即時排出一張完整的紫微斗數命盤。

| 項目 | 說明 |
|------|------|
| **十四主星** | 紫微、天機、太陽、武曲、天同、廉貞、天府、太陰、貪狼、巨門、天相、天梁、七殺、破軍 |
| **輔星** | 左輔、右弼、文昌、文曲、天魁、天鉞 |
| **煞星** | 地空、地劫、祿存、擎羊、陀羅、火星、鈴星、天馬 |

### 運限六項
| 運限 | 說明 |
|------|------|
| **大限** | 10 年運勢 |
| **小限** | 1 年個人限制 |
| **流年** | 當年運勢 |
| **流月** | 当月運勢 |
| **流日** | 當日運勢 |
| **流時** | 當辰運勢 |

### 四化
祿、權、科、忌 — 依天干推算各星曜的四化狀態。

## 技術選型

### 計算引擎：[iztro](https://github.com/SylarLong/iztro)

MIT 開源 JavaScript 庫，支援：
- 陽曆 / 農曆雙輸入
- 性别參數
- 12 宮星盤數據（含所有主星、輔星、煞星）
- 運限：大限、小限、流年、流月、流日、流時
- 四化判斷（祿、權、科、忌）
- 三方四正分析
- 多語言輸出（繁中、簡中、英文等）

```js
import { astro } from 'iztro';
const chart = astro.bySolar('2000-8-16', 2, '男', true, 'zh-TW');
```

### 前端架構
- **單一 HTML 檔案**（含 CSS + JS）
- `iztro` via CDN (`cdn.jsdelivr.net`)
- 無後端、無框架依賴

## 排盤邏輯摘要

### 1. 安十二宮
```
寅宮 → 順數至生月地支 → 逆數至出生時辰 = 命宮
```

十二宮順時針排列：命宮 → 父母 → 福德 → 田宅 → 官祿 → 僕役 → 遷移 → 疾厄 → 財帛 → 子女 → 夫妻 → 兄弟

### 2. 定五行局
依生年天干 + 命宮地支查表確定（水二局、木三局、金四局、土五局、火六局）

### 3. 安紫微星
```
倍數 = ceil(出生日 / 五行局數)
差數 = 倍數 × 五行局數 - 出生日

寅起步數 = 奇數差 → 倍數 - 差數
           偶數差 → 倍數 + 差數
```

### 4. 安十四主星
紫微星定位後，逆時針依序安置其餘十三顆主星。天府星另起（南斗），順時針安置。

### 5. 安輔煞星
依出生月份、時辰、年干照表填格。
